Estudo de semantica

Negrito

Nesta frase temos um termo em negrito usando a tag <b> (é uma tag não semantica)

Para termo uma tag semantica usamos a tag <strong>

Italico

Agora vamos, para italico termo em italico usamos a tag <i> (não semantico)

Agora o italico semantico para isso usamos a tag <em>

Marca texo

Para marcar texto usamos a tag <mark>

Texto grande e pequeno

Grande usamos a tag <big> e pequeno usamos a tag <smal>

Texto deletado

Texto apagado usamos a tag <del>

Texto inserido

Texto inserido usamos a tag <ins>

Texto sobrescrito

Texto sobrescrito exxemplo x20+3 usamos a tag <sup>

Texto subscrito

Texto subscrito Exemplo H2O usamos a tag <sub>
